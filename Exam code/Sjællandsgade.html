Sjællandsgade District

Sjællandsgade is an old street in Aarhus, it started
in the beginning of the 1800s. The street used to
be working class district, a number of merchants
and other shops were located here. But in the
1970s it was starting to change into a more
thriving area. Most of the shops are gone now,
but you might still be able to find a few. It is a
charming street with old architecture.

All the houses have different
colors and it is a very unique
street that is worth going to.
